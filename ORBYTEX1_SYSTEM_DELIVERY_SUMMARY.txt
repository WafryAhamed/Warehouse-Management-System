🎯 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. Setup Database:
   psql> CREATE DATABASE orbytex1;
   psql> CREATE USER orbytex_user WITH PASSWORD '2001';
   psql> GRANT ALL PRIVILEGES ON DATABASE orbytex1 TO orbytex_user;

2. Start Backend:
   cd OrbyteX1-Backend
   mvn spring-boot:run
   (Wait for "Started OrbyteX1Application")

3. Start Frontend:
   cd OrbyteX1-Frontend
   mvn javafx:run
   (Frontend window should open)

4. Login:
   Username: admin
   Password: password
   (Click "Use Demo Credentials" button for easy testing)

5. Test Dashboard:
   - View real-time KPI cards
   - Navigate through dashboards
   - Logout and re-login
