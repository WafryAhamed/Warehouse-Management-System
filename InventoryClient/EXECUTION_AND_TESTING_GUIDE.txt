## TROUBLESHOOTING

### Issue: Maven not found
**Solution**:
```bash
# Check Maven installation
mvn -version

# If not found, ensure Maven is in PATH
```

### Issue: Java compilation fails
**Solution**:
```bash
# Check Java version
java -version

# Ensure Java 21+ is installed
```

### Issue: FXML files not found
**Solution**:
```bash
# Rebuild resources
mvn clean compile
```

### Issue: Application won't run
**Solution**:
```bash
# Clear dependencies and rebuild
mvn clean dependency:resolve
mvn javafx:run
```
