### BUG #1: Dialog Cancel Exception (CRITICAL) ✅ FIXED
**File**: InventoryController.java:163
**Severity**: CRITICAL
**Issue**: `showAndWait().get()` throws NoSuchElementException when user cancels dialog
**Root Cause**: `.get()` on Optional throws exception if empty (dialog cancelled)
**Fix Applied**:
```java
// BEFORE (BUGGY)
if (confirmation.showAndWait().get() == ButtonType.OK) {

// AFTER (FIXED)
if (confirmation.showAndWait().filter(response -> response == ButtonType.OK).isPresent()) {
```
**Impact**: Prevents application crash when canceling delete confirmation
**Tested**: ✅

### BUG #6: JSON Integer Parsing Doesn't Handle Negatives (MEDIUM) ✅ FIXED
**File**: JsonUtil.java:96
**Severity**: MEDIUM
**Issue**: Regex pattern `\\d+` doesn't match negative numbers
**Root Cause**: Missing negative sign in regex pattern
**Fix Applied**:
```java
// BEFORE (BUGGY)
Pattern pattern = Pattern.compile("\"" + key + "\"\\s*:\\s*(\\d+)");

// AFTER (FIXED)
Pattern pattern = Pattern.compile("\"" + key + "\"\\s*:\\s*(-?\\d+)");
```
**Impact**: Can now parse negative integer values from JSON
**Status**: ✅ FIXED
