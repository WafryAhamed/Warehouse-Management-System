### Current Limitations
1. **Mock Authentication**: Hardcoded credentials (development only)
2. **Incomplete Services**: SalesService and SupplierService are placeholders
3. **No Pagination**: Product list not paginated for large datasets
4. **No Caching**: API responses not cached locally
5. **No Offline Mode**: Requires API connectivity
6. **Memory-Based Tokens**: Tokens lost on application restart
